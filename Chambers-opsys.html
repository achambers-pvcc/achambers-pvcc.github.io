Operating Systems Terminology

Term defintions
Operating system the software that supports a computer's basic functions, such as scheduling tasks, executing applications, and controlling peripherals
Job queue is a data struture maintained by job schedular software containing jobs to run
FIFO the acronym for first in,first out a common method for organizing a data buffer as a queue
interactive processing the process of interacting directly with a program and data for immediately modifying information
multiprogramming the operation of a computer system by several users for different operations at the same time
time-sharing a form of parallel processing in which several programs run at the same time on a uniprocessor system
multiprogramming is a parallel processing in which several programs run at the same time on a uniprocessor system
application software is a computer program designed to carry out a specific task other than one relating to the operation of the computer itself
utility software designed to help analyze, configure, optimize or maintain a computer
graphical user interface designed to help analyze,configure,optimize or maintain a computer graphical user interface(GUI)
kernel core component of an operating system and serves as the main interface between the computer's physical hardware and processes running on it
device driver a software program that controls a specfic hardware device attached to a computer
file manager used for file maintannace operations
memory manager the process of controlling and coordinating a computer's main memory
virtual memory a common technique used in a computer's operating system
boot strapping to load a program into a computer using a much smaller initial program to load in the desired program
ROM a flash memory chip that contains a small amount of non-volatile memory
deadlock is when no member of some group of entities can proceed because each waits for another member, including itself
spooling is a process which data is temporarily held to used and executed by a device, program or the system super user (or administrator)
multi-factor authentication is an electronic authentication method in which a user is granted access to a website or application after successfully presenting two or more pieces of evidence to an authentication mechanism
